Website Style Guide

This is the style guide to the website. The colours were chosen to reflect simplicity and sophistication which are in line with the character's personality. As my character is a psychology colours for the site were chosen based on colour psychology. The main colour is blue which is said to reflect serenity, tranquility, and calmness.

Overview Of Wireframes

These are the old wireframes based on initial plans for the website.

Homepage Wireframe

The annotated home page wireframe has not changed much as compared to the original. The main adjustment for the wireframe is that a search bar has been installed to allow site visitors to search the website for improved user experience.

AboutPage Wireframe

The annotated About page wireframe has been adjusted to include a footer so that the user may access the necessary information about the legalities of the site even when they are on this page. Furthermore, the search bar has also been included for improved user experience.

Blog Post Wireframe

No changes have been made to the blog post wireframe for the desktop view.

Contact Page Wireframe

No changes have been made to the contact page wireframe for desktop view.

Tablet And Mobile View Wireframe

To ensure further quality user experience I aim to turn the website into a responsive website that allows all content to adjust and fit properly on all screen sizes.
